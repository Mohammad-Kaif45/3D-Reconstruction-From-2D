API Endpoints

Image Upload

POST /api/upload/

Upload a 2D image for 3D reconstruction.

Request Body:
image_file File Required
model_name String Optional

Get Model Status

GET /api/status/{task_id}/

Check the status of a previously submitted 3D reconstruction task.

Query Parameters:
task_id String Required

Download Model

GET /api/download/{task_id}/

Download the final 3D model file after processing.

Query Parameters:
task_id String Required
